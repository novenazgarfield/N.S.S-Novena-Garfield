NEXUS智能仪表板系统介绍

NEXUS是一个现代化的智能仪表板系统，具有以下特点：

1. 用户界面设计
- 采用深色主题设计，提供优雅的视觉体验
- 响应式布局，支持桌面和移动设备
- 侧边栏导航，方便快速访问各个功能模块

2. 核心功能模块
- 项目介绍：展示系统概览和主要特性
- 技术栈：显示使用的技术和工具
- 下载页面：提供各平台的下载链接
- RAG聊天：集成智能问答系统

3. RAG智能问答系统
- 支持文档上传和分析
- 基于向量检索的智能问答
- 支持多种文件格式：PDF、Word、图片等
- 实时聊天界面，支持上下文对话

4. 技术特性
- 前端：HTML5、CSS3、JavaScript
- 后端：Python Flask API
- 数据处理：文档解析、文本检索
- 用户体验：加载状态、错误处理、响应式设计

5. 使用方法
- 访问主页查看系统介绍
- 上传文档到RAG系统
- 通过聊天界面提问
- 获得基于文档内容的智能回答

NEXUS系统致力于提供高效、智能的信息管理和问答体验。
